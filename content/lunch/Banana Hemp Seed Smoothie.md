This banana-based smoothie with hemp seed and almond butter is a hearty way to start the day and is paleo-friendly! 

{{< boldheading >}}

{{< checkbox "2  bananas" >}}

{{< checkbox "2 tablespoons almond butter" >}}

{{< checkbox "2 cups water" >}}
